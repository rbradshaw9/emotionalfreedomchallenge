# Logo Directory

Place the logo files for The Emotional Freedom Challenge in this directory.

Recommended logo specifications:
- Primary logo (horizontal): SVG or PNG format
- Minimum dimensions: 300px wide
- Transparent background preferred
- Include both light and dark versions if needed

File naming suggestions:
- `logo.svg` or `logo.png` - Primary logo
- `logo-dark.svg` or `logo-dark.png` - Dark version (if needed)
- `favicon.ico` - Browser favicon
- `apple-touch-icon.png` - Apple touch icon (180x180px)

Once added, update the Header component at `/app/components/Header.tsx` to reference the logo path.

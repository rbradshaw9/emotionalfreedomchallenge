# The Emotional Freedom Challenge

A Next.js 14+ App Router project for The Emotional Freedom Challenge by Dr. Paul Jenkins.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This project is designed for deployment on Vercel.

## Domain

This site will be deployed to: challenge.drpauljenkins.com

## Pages

- `/` - Registration page
- `/vip` - VIP upsell page
- `/thank-you` - Thank you / next steps page
- `/replay` - Replay hub for Days 1-5
- `/privacy` - Privacy policy
- `/terms` - Terms of use
